How to Play

The Basics

Triple Triad is played on a 3×3 grid. Each player has 5 cards. Players take turns placing one card on an empty cell. The goal: own the majority of cards (6 or more out of 10) when the board is full.

Card Ranks

Every card has 4 rank numbers — one per side: Top, Right, Bottom, Left. Ranks range from 1 to 10 (10 is shown as A). When you place a card adjacent to an opponent's card, compare the touching sides. If your rank is higher, you capture their card (it flips to your colour).

Special Rules

Different regions play with different rules. These change how captures work:

RuleEffect
OpenYou can see the opponent's hand before playing.
SameIf your card's touching ranks equal the adjacent cards' ranks on 2+ sides, capture those cards. Triggers Combo.
Same WallBoard edges count as rank A (10) for Same checks.
PlusIf the sums (your rank + their rank) are equal on 2+ sides, capture those cards. Triggers Combo.
ComboCards captured by Same/Plus cascade — any adjacent weaker card is also captured.
ElementalBoard cells have elements. Matching element: +1 to all ranks. Mismatched: −1.
RandomYour 5 cards are randomly chosen from your deck.
Sudden DeathOn a draw, the game replays with the board as-is until someone wins.

Trade Rules

What happens to cards after a match:

RuleStakes
OneWinner takes 1 card from the loser.
DiffWinner takes 1 card per point of score difference.
DirectEach player keeps the cards in their colour at game end.
AllWinner takes ALL of the loser's played cards.
Tip: In-game tutorial popups explain each rule the first time you encounter it. You can revisit all tutorials from the Guide tab at any time.

Rule Spreading

After Chapter 5, dueling in a new region can carry rules from your last-played region. Rules can spread to or be abolished from regions over time. The Queen of Cards in Esthar City lets you deliberately manipulate rules across any region for a gil fee.

Progression

  • Win duels to earn gil and capture opponent cards
  • Buy cards from shops in each town
  • Enter tournaments for rare card prizes
  • Complete quests to unlock new regions and earn rewards
  • Build your deck — save multiple deck loadouts for different strategies

Settings

Accessible from the title screen. Customise your experience:

  • Text Scale (70%–150%) — adjust all game text with a live preview
  • Card Overlay (0–150%) — control the size of rank numbers on cards, or turn them off entirely for a clean art-only look

Running on Web (Docker)

The web version runs entirely via Docker — no Node.js, no local dependencies needed.

Prerequisites

  • Docker Desktop (Windows/Mac) or Docker Engine (Linux)
  • Git to clone the repository

Quick Start

  1. Clone the repository
    git clone https://github.com/Alex-ReachIndustries/TripleTriad.git cd TripleTriad
  2. Start with Docker Compose
    docker compose up -d --build

    This builds and starts both the frontend dev server and the WebSocket backend for 2P duels.

  3. Open in your browser
    http://localhost:5173

    The game loads immediately. Your save data is stored in your browser's localStorage.

  4. To stop
    docker compose down
Note: The 2P Duel mode requires the backend WebSocket server (included in docker-compose). Single-player World Mode works entirely in the browser with no backend needed.

Production Build

To create an optimised static build:

docker run --rm -v "$(pwd)/frontend:/app" -w /app node:22-slim sh -c "npm ci && npm run build"

The output in frontend/dist/ can be served by any static file server (Nginx, Caddy, GitHub Pages, etc.).

Installing on Android

The Android APK is available on the GitHub Releases page. Since it's not from the Play Store, you'll need to allow installation from unknown sources.

Download

  1. Get the APK

    Go to the GitHub Releases page and download TripleTriad.apk from the latest release (v1.0.0).

    https://github.com/Alex-ReachIndustries/TripleTriad/releases
  2. Transfer to your device

    If you downloaded on a PC, transfer the APK to your phone via USB cable, cloud storage, or email it to yourself.

Enable Unknown Sources

Important: Android blocks apps from outside the Play Store by default. You need to temporarily allow installation from the source you're using (browser, file manager, etc.).

Android 8.0+ (Oreo and newer)

  1. Open the APK file on your phone (tap the download notification or find it in your file manager).
  2. Android will prompt: "For your security, your phone is not allowed to install unknown apps from this source."
  3. Tap Settings on the prompt. This opens the per-app install permission screen.
  4. Toggle "Allow from this source" (or "Install unknown apps") to ON.
  5. Go back and tap Install.
  6. Optional but recommended: After installing, go back to Settings > Apps > Special access > Install unknown apps and toggle the permission back OFF for security.

Android 7.0 and older

  1. Go to Settings > Security.
  2. Enable "Unknown sources" (this is a global toggle on older Android).
  3. Open the APK and tap Install.
  4. Disable "Unknown sources" again after installation.

After Installation

  • Find Triple Triad in your app drawer and tap to launch.
  • The game runs entirely offline — no internet connection needed for single-player.
  • Use the Android back button to navigate (tabs → world → title screen).
  • Your save data persists between sessions automatically.
